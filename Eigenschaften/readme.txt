Eigenschaftensets für den Import in Archicad (Optionen>Eigenschaftenmanager). Die Archicad-Eigenschaften sind für das Objekt selbst nicht notwendig, werden aber ggf. in den Türlisten benötigt. Die Entscheidung, ob eine Eigenschaft als Archicad-Eigenschaft oder als GDL-Parameter angelegt wurde, ist durch die Relevanz für Geometrie bzw. Logik/Konfiguration/Information für das Skript determiniert.   

Die Versionsnummer zeigt an für welches Archicad das Set entwickelt wurde. Die Kompatibilität mit anderen Versionen kann evt. dennoch möglich sein, wurde aber nicht getestet.
